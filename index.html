학령 인구 데이터 분석

공공 데이터를 이용해 대한민국 학령 인구의 변화를 Tableau 소프트웨어를 통해 데이터 시각화를 함. 2017년부터 2028년까지 인구 변화 추이를 지역 위치별로 표현함.

KT 로밍 데이터 분석

KT에서 공개한 로밍 데이터를 기반으로 국내 관광객이 집중적으로 모이는 지역과 출입 경로와 이동 동선을 분석함. Excel과 Tableau를 사용하였으며 동시 다발적으로 지역에 일어나는 call 현황으로 집중 지역을 파악함.

편의점 고객 시뮬레이션

Arena 소프트웨어를 활용해 편의점에 이용 고객 데이터로 시뮬레이션을 만들고 대기행렬과 입,퇴장을 애니메이션화 하였으며 분석 그래프를 결과적으로 할 수 있었음.
